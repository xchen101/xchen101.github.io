A lightweight, self-hosted photo blog built with Jekyll. Minimal interface, fast loading, designed for street photography.

The idea was simple: a place to post photos with minimal friction. No CMS, no database, no JavaScript frameworks. Just markdown files, images, and a clean grid that stays out of the way.

Built with Jekyll and hosted on GitHub Pages. The layout uses a responsive CSS grid that adapts from three columns on desktop to a single column on mobile. Images are lazy-loaded and served in multiple sizes for performance.

The design is intentionally restrained — no filters, no overlays, no social sharing buttons. Just the photo, a title, and a date.
